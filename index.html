01
Gene Expression & Epigenetics
Exam 1
Core idea: Every somatic cell carries the same DNA. Differences in cell identity arise entirely from which genes are expressed — controlled by transcription factors, enhancers, and heritable epigenetic marks.

Regulatory Elements

Promoter — at TSS; TATA box. Enhancer — distant; bound by TFs to boost transcription. Silencer — represses. Insulator — blocks enhancer-promoter contact.

TF bindingChromatin

Epigenetic Mechanisms

DNA methylation (CpG → silencing). Histone acetylation → open/active. H3K4me3 = active; H3K27me3 = repressed (Polycomb). All heritable without altering DNA sequence.

HeritableNo sequence change

UAS/Gal4 (Drosophila)

Gal4 drives any gene under tissue-specific promoter via UAS binding site. Find it (GFP reporter), Move it (ectopic expression), Lose it (UAS-RNAi knockdown).

DrosophilaGenetic tool

Meiosis & Aneuploidy

MI non-disjunction → all gametes aneuploid. MII error → half normal. Use heterozygous markers to identify parent of origin. Prenatal tests: CVS, amniocentesis, cfDNA (non-invasive).

Non-disjunction
02
Proteins, Antibodies & Lab Methods
Exam 1
Core idea: Protein structure dictates function. Antibodies exploit specific binding to let us visualize, isolate, and quantify proteins — matching the technique to the biological question is everything.

Western Blot

Lyse → SDS-PAGE (size) → transfer → primary Ab → secondary-HRP → chemiluminescence. Answers: size and relative abundance of a protein.

Immunofluorescence

Fix → primary Ab → fluorescent secondary → confocal. Answers: subcellular localization. Co-localization ≠ interaction. BiFC: fluorescence only if proteins physically contact.

Co-IP

Pull down protein A → western for B. Answers: physical interaction in cells. Used to confirm CDK–cyclin binding; test if mutant cyclin still interacts with CDK.

Protein Structure

1° aa sequence → 2° α-helix/β-sheet → 3° folded shape (H-bonds, disulfide, van der Waals, hydrophobic) → 4° multi-subunit. Enzymes lower activation energy; allosteric ligands change conformation.

03
Membranes, Transport & Cell Architecture
Exam 2
Core idea: Membranes define compartments. Selective transport proteins control what moves between them. Protein targeting sequences act as molecular zip codes.
Membrane Transport Rules

Small nonpolar (O₂, CO₂) → free diffusion. H₂O → aquaporins or slow diffusion. Ions + large polar → channels (passive, fast, no conformational change) or carriers (conformational change; can be active or passive). Blocking Na⁺/K⁺ ATPase ↑[Na⁺]ᵢ → collapses Na⁺ gradient → impairs Na⁺-linked cotransporters.

Protein Targeting
  • Signal peptide → ER (co-translational)
  • NLS → nucleus (importins + Ran-GTP gradient)
  • Transit peptide → mitochondria/chloroplasts (post-translational, Hsp70-assisted)
  • ER → Golgi → plasma membrane via SNARE vesicle fusion
  • TM orientation set at ER — never changes after insertion

⚡ Cellular Respiration

Glycolysis (cytoplasm, net 2 ATP) → Citric acid cycle (matrix, NADH/FADH₂) → ETC + ATP synthase (inner membrane, ~32 ATP via chemiosmosis). Anaerobic: lactate (muscle) recycles NADH → NAD⁺ to sustain glycolysis.

🕸️ Cytoskeleton Summary

Actin (polar, treadmilling) → motility, shape. Microtubules (polar, dynamic instability) → division, transport. IFs (non-polar) → mechanical integrity. Septins → cytokinesis ring. Motor proteins use ATP hydrolysis.

05
Cancer Biology
★ Primary Focus
Core idea: Cancer is disrupted cell fate — cells that should stop, differentiate, or die keep proliferating. It evolves by Darwinian natural selection within the body.
Oncogenes — "gas pedal"

Proto-oncogene → oncogene via gain-of-function (mutation, amplification, translocation). Dominant — one mutant allele sufficient. Key examples: RAS (constitutive GTPase), MYC (overexpressed TF), HER2 (amplified RTK).

Tumor Suppressors — "brakes"

Loss-of-function mutations. Recessive — both alleles must be lost (two-hit hypothesis, Knudson). Key examples: RB1 (G1 checkpoint), TP53 (DNA damage response), APC (Wnt inhibitor), BRCA1/2 (DNA repair).

Benign vs. Malignant

Benign: encapsulated, slow-growing, no metastasis, resembles origin tissue. Malignant: invasive, metastatic, anaplastic. Metastasis is the defining feature of malignancy and the main cause of cancer mortality.

Definition

Cancer Evolution

Sequential mutations accumulate; cells with growth advantage clonally expand (somatic Darwinian selection). Classic: colorectal cancer — APC → KRAS → SMAD4 → TP53. Intratumor heterogeneity drives therapy resistance.

DarwinianClonal

Cancer Stem Cells

Self-renewing subpopulation. CD44⁺/CD24⁻ in breast cancer. Drive initiation, maintenance, and relapse (quiescent → survive chemo → regrow tumor). Plastic — non-CSCs can dedifferentiate.

StemnessRelapse

Tumor Angiogenesis

Tumors >1–2 mm need new vessels. Hypoxia → HIF-1α → VEGF → endothelial sprouting. Negative feedback: O₂ delivery → HIF-1α degradation → VEGF drops. DLL4/Notch (tip vs. stalk) limits excess sprouting.

VEGFFeedback
06
Cell Signaling, Cell Cycle & Specification
Exam 3
Core idea: Cells interpret signals through receptor cascades and execute fate decisions via the cell cycle. The CDK/cyclin machinery is the physical gate to division — and cancer breaks it repeatedly.

GPCR Signaling

Ligand → 7-pass receptor → Gα-GTP release → adenylyl cyclase (cAMP → PKA) or PLC (IP₃ + DAG → PKC). Terminated by Gα GTPase. Examples: β-adrenergic, rhodopsin.

cAMPIP₃/DAG

Enzyme-Coupled (RTK)

Ligand → dimerization → autophosphorylation → docking for RAS, PI3K, PLC-γ → MAPK or Akt pathways. Terminated by phosphatases + receptor endocytosis. Examples: EGF-R, insulin-R.

RAS-MAPKPI3K/Akt

CDK/Cyclin & Cell Cycle

G1: Cyclin D/CDK4/6 → phospho-Rb → releases E2F → S-phase genes. S: Cyclin E/CDK2. M: Cyclin B/CDK1. Mutations overactivating cyclins or inactivating Rb = cancer.

Cancer linkRb/E2F

Cell Specification

Autonomous: fate from cytoplasmic determinants (C. elegans P granules via PAR proteins). Conditional: fate from neighbor signals (vertebrate neural induction). Test: isolate cell — does it develop same as in embryo?

PAR proteinsInduction

Epistasis in Signaling

Epistatic gene/protein acts downstream (its phenotype wins in double mutant). Constitutively active = "always on." Dominant negative = "pathway blocked." Same logic as genetics crosses — APC upstream of β-catenin, RAS upstream of RAF.
